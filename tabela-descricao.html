Descrição de Tags

TAG DESCRIÇÃO
TABLE Representa uma tabela em HTML
TH Representa cada cabeçalho da tabela
TR Representa cada linha de uma tabela
TD Representa cada célula da tabela
THEAD Para representar o cabeçalho da tabela
TBODY Para representar o corpo da tabela
TFOOT Para representar o rodapé da tabela
ALIGN Alinhar alguns elementos
BORDER Adiciona uma borda na tabela como um todo
CAPTION Definir um título
ROWSPAN É o número de linhas que você quer que sejam ocupadas
COLSPAN Permite que uma célula ocupe várias colunas
WIDTH Determina a largura da área de conteúdo de um elemento
